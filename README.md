## About the Project

NASA Exoplanets AI is a machine learning project developed as part of the **2025 NASA Space Apps Challenge**.  
The project focuses on automatically identifying exoplanets using open-source datasets from NASA missions such as **Kepler, K2, and TESS**.  

By leveraging artificial intelligence and data analysis techniques, this project aims to reduce the need for manual analysis and help researchers and enthusiasts explore exoplanet data more efficiently.

---

## Project Objectives

This project was created to:

- Automate the identification of exoplanets from NASA datasets.
- Classify signals as confirmed exoplanets, candidates, or false positives.
- Explore how AI/ML can assist astronomers in analyzing large-scale space data.
- Provide a reproducible and easy-to-follow notebook for learning and experimentation.

---

## Features

- **Data Preprocessing**
  - Cleans and prepares raw NASA exoplanet datasets.
  - Handles missing values and feature selection.

- **Machine Learning Model**
  - Trains an ML model to classify exoplanet signals.
  - Evaluates performance using accuracy and other metrics.

- **Data Analysis & Visualization**
  - Visualizes trends and patterns in exoplanet data.
  - Helps interpret model predictions.

- **Reproducible Notebook**
  - Fully implemented in a Jupyter/Google Colab notebook.
  - Easy to run and modify for experimentation.

---

## Dataset

- **Source:** NASA Open Exoplanet Archives  
- **Missions Used:** Kepler / K2 / TESS  
- **Data Includes:**
  - Orbital period
  - Transit duration
  - Planetary radius
  - Stellar and orbital parameters
  - Classification labels (confirmed, candidate, false positive)
